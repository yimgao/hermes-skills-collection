# Local Competitive Analysis

> Analyze local competitors for a specific industry in a given city/location. Automatically detects location via IP geolocation if not provided. Produces a structured market report.

---

## Overview

This skill helps you research local competitors for a specific type of business (e.g., Ramen, Coffee Shop, Coworking Space) in a specific city or neighborhood. It automatically detects the user's location via IP geolocation if no location is given, then searches the web for top competitors, gathers ratings and reviews, and synthesizes a structured market report.

The output covers:
- Local market overview
- Top 5 competitors with ratings and review counts
- SWOT analysis (Strengths / Weaknesses / Opportunities / Threats)
- Market gap identification
- Strategic recommendations for opening a new business

---

## When to Use

- User asks: *"Analyze the Ramen market near me"*
- User asks: *"Research coffee shops in Shanghai — who are the top competitors?"*
- User asks: *"I want to open a bakery in Tokyo. What's the competitive landscape?"*
- User asks: *"What's missing in the coworking space market of my city?"*
- User asks: *"Do a local competitive analysis for [industry] in [location]"*

**Parameters the user provides:**
| Parameter | Required | Description | Example |
|-----------|----------|-------------|---------|
| `industry` | ✅ Yes | Type of business / industry to analyze | "Ramen", "Coffee Shop", "Coworking Space" |
| `location` | ❌ No | City or neighborhood | "Shibuya, Tokyo", "Shanghai" (auto-detected if omitted) |

## Core Workflow

### Step 1: Resolve Location

If the user **did not** provide a `location`, auto-detect it:

```
1. Fetch: https://ipinfo.io/json (or similar IP geolocation API)
2. Extract: city, region, country
3. Use the city name as the location
```

If the user **did** provide a location, use that directly.

**Note:** The API response includes fields like `city`, `region`, `country`, and `loc` (coordinates). Use `city` as the primary location, optionally appending `region` or `country` for disambiguation.

### Step 2: Search for Competitors

Use `web_search` or the browser to find the top 5 local competitors. Use multiple search queries to ensure comprehensive results:

**Primary search:**
```
"top {industry} in {location}" OR "best {industry} near {location}" OR "{location} {industry} rankings"
```

**Fallback / enrichment searches:**
```
"{industry} {location} reviews"
"{industry} {location} recommendations"
"{location} best rated {industry}"
```

For each competitor found, extract:
- **Name** (business name)
- **Rating** (out of 5, where available)
- **Review count** (number of reviews)
- **Source** (Google Maps, Yelp, Dianping, TripAdvisor, etc.)
- **Brief description / specialty** (what they're known for)

Select the **top 5** most relevant / highest-rated competitors.

> 💡 **Data collection tip:** If search results don't show clear ratings, try browsing Google Maps directly (e.g., `google.com/maps/search/{industry}+{location}`) or local review sites specific to the country (e.g., Dianping/大众点评 for China, Tabelog/食べログ for Japan, Yelp for US).

## Common Pitfalls

| # | Problem | Solution |
|---|---------|----------|
| 1 | **IP geolocation may give a broad area (not neighborhood).** | Use the city name; if the user wants a specific neighborhood, ask them. |
| 2 | **Search results may not show ratings directly.** | Try browsing Google Maps URL directly, or use location-specific review platforms (Dianping for China, Tabelog for Japan, Yelp for US). |
| 3 | **Review counts may be from different platforms.** | Note the source of each rating/review count. Don't mix Google Maps ratings with Yelp ratings without calling it out. |
| 4 | **Competitor info may be outdated.** | Check publication dates of any articles/reviews. Favor recent (last 6 months). |
| 5 | **Industry terms are ambiguous.** | "Ramen" is clear, but "Coffee Shop" and "Café" may overlap. Use both terms in search. |
| 6 | **No results for a specific niche.** | Broaden the search: e.g., "Japanese food" instead of "Ramen" | if "Ramen" alone returns too little. |
